# Premium Romantic Storybook

## Setup Instructions

### Audio File
Place a page flip sound effect as `sounds/flip.mp3`

You can:
1. Download a free page flip sound from freesound.org
2. Or use this online generator: https://sfxr.me/
3. Or record your own page flip sound

### Images
Add romantic images to the `images/` folder:
- images/page1.jpg
- images/page2.jpg
- images/page3.jpg
- images/page4.jpg

Recommended: Use soft, romantic, pastel-toned images.

- Click Next/Previous buttons

- Audio plays after first interaction
