✅ Mobile responsive

### Optional Enhancements:

#### To Add Images:
1. Open `generate-images.html` in browser
2. It will download 4 placeholder images
3. Move them to the `images/` folder
4. Refresh the storybook

#### To Add Page Flip Sound:
1. Visit: https://freesound.org/search/?q=page+flip
2. Download a short page flip sound (0.5-1 second)
3. Save as `sounds/flip.mp3`
4. Refresh the storybook

### Alternative for Audio:
Use this free sound:
https://freesound.org/people/f4ngy/sounds/240776/
